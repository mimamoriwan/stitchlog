# タスク: PDFにフレンチノットレイヤーを反映

## 前提確認
generatePdf.ts を開いて以下を確認してから実装する:
- バックステッチ描画ブロックの変数名（offsetX・offsetY・cellSize相当）
- layers.frenchKnot の型（FrenchKnot型のフィールド名: x, y, colorCode, wraps等）
  → packages/types/src/index.ts を確認する
変数名・フィールド名は必ず既存コードに合わせる。

## 実装仕様
フレンチノットは「小さい塗りつぶし円」としてグリッド上に描画する。

```typescript
// 描画ルール
for (const knot of patternData.layers.frenchKnot) {
  const cx = offsetX + knot.x * cellSize + cellSize / 2;
  const cy = offsetY + knot.y * cellSize + cellSize / 2;
  const radius = cellSize * 0.25; // マス目の25%サイズ

  // 色設定（isCatchlight: true → 白、それ以外 → 黒）
  if (knot.isCatchlight) {
    doc.setFillColor(255, 255, 255);
    doc.setDrawColor(180, 180, 180); // 白丸の輪郭をうっすら表示
  } else {
    doc.setFillColor(0, 0, 0);
    doc.setDrawColor(0, 0, 0);
  }

  doc.setLineWidth(0.1);
  doc.circle(cx, cy, radius, 'FD'); // F=塗りつぶし D=輪郭
}
```

## 挿入位置
バックステッチ描画ブロックの直後に挿入する。
（描画順：グリッド → バックステッチ → フレンチノット）
