# タスク: フレンチノットレイヤーの実装（目のハイライト・点）

## 背景
現在 layers.frenchKnot が空配列のまま。
フレンチノットは目の光・小さな点を表現する。
バックステッチと違い「線」ではなく「点」として記録する。

## 実装場所
packages/conversion-engine/src/frenchKnotDetector.ts を新規作成。

## 検出ロジック
「周囲より極端に明るい孤立した点」= フレンチノット候補。

Step 1: グリッドサイズにリサイズ + グレースケール化
Step 2: 各ピクセルの輝度を取得
Step 3: 3×3近傍の平均輝度より 60以上明るい孤立点を検出
Step 4: 検出点が隣接していない（孤立している）ことを確認
Step 5: FrenchKnotPoint として登録

## frenchKnotDetector.ts の内容

```typescript
import sharp from 'sharp';
import type { FrenchKnotPoint } from '@stitchlog/types';

export async function detectFrenchKnots(
  imageBuffer: Buffer,
  gridWidth: number,
  gridHeight: number,
): Promise<FrenchKnotPoint[]> {
  // バックステッチと同様に半サイズで処理
  const halfW = Math.ceil(gridWidth / 2);
  const halfH = Math.ceil(gridHeight / 2);

  const { data, info } = await sharp(imageBuffer)
    .resize(halfW, halfH, { fit: 'fill' })
    .grayscale()
    .raw()
    .toBuffer({ resolveWithObject: true });

  const { width, height } = info;
  const knots: FrenchKnotPoint[] = [];

  for (let y = 1; y < height - 1; y++) {
    for (let x = 1; x < width - 1; x++) {
      const center = data[y * width + x];

      // 3×3近傍の平均輝度（中心を除く8近傍）
      let neighborSum = 0;
      for (let ky = -1; ky <= 1; ky++) {
        for (let kx = -1; kx <= 1; kx++) {
          if (ky === 0 && kx === 0) continue;
          neighborSum += data[(y + ky) * width + (x + kx)];
        }
      }
      const neighborAvg = neighborSum / 8;

      // 近傍より60以上明るく、かつ輝度200以上（十分明るい）の孤立点
      if (center - neighborAvg >= 60 && center >= 200) {
        knots.push({
          x: x * 2,  // 座標を2倍にスケールアップ
          y: y * 2,
          colorCode: 'DMC-blanc',
          plyCount: 1,
        });
      }
    }
  }

  // 上限300点（過検出防止）
  return knots.slice(0, 300);
}
```

import { detectFrenchKnots } from '@stitchlog/conversion-engine';

const frenchKnots = await detectFrenchKnots(
  imageBuffer,   // バックステッチと同じ変数名を使う
  patternData.stitchCountX,
  patternData.stitchCountY,
);
patternData.layers.frenchKnot = frenchKnots;
