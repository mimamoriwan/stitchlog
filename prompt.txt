# タスク: PDFにバックステッチレイヤーを反映

## 背景
generatePdf.ts は現在クロスステッチグリッドのみ描画している。
layers.backStitch のセグメントをグリッド上に線として重ねて描画する。

## 仕様

### 線の描画ルール
- 各セグメント（fromX/fromY → toX/toY）を jspdf の line() で描画
- plyCount === 2（強エッジ）→ 線幅 0.6pt
- plyCount === 1（弱エッジ）→ 線幅 0.3pt
- colorCode が "DMC-310"  → setDrawColor(0, 0, 0)
- colorCode が "DMC-3799" → setDrawColor(43, 43, 43)
- 上記以外               → setDrawColor(0, 0, 0)（フォールバック）

### 座標計算
グリッドの描画開始 X/Y オフセットと cellSize を使い、
各マス目の中央を通るよう計算する（fromX * cellSize + offsetX + cellSize/2 など）。
既存のグリッド描画変数をそのまま流用すること。変数名は必ず既存コードを確認してから使う。

## 実装場所
apps/web/src/lib/generatePdf.ts の「グリッド描画ブロックの直後」に追加。
既存のグリッド描画ループとオフセット変数を確認してから実装すること。

## 制約
- jspdf の setDrawColor / setLineWidth / line() のみ使用
- 外部ライブラリ追加なし
- グリッド描画後にバックステッチを上書きする順序を守る（透過でないため）
- エラーが出たら止まって報告する

## 動作確認
1. pnpm dev でサーバー起動
2. 写真をアップロードして変換
3. PDFダウンロード → Acrobat/プレビューで輪郭線が表示されることを確認
