# タスク: バックステッチレイヤーの実装（輪郭・ヒゲ）

## 背景
現在 layers.backStitch が空配列のまま。
バックステッチはヒゲ・輪郭・細い線を表現する最重要レイヤー。

## 実装方針
sharp に Canny エッジ検出は存在しないため、
Sobel演算子（convolution）でエッジ検出を実装する。
OpenCV.js などの外部ライブラリは追加しない。

## 実装場所
packages/conversion-engine/src/backstitchDetector.ts を新規作成。

## backstitchDetector.ts の内容

```typescript
import sharp from 'sharp';
import type { BackStitchSegment } from '@stitchlog/types';

export async function detectBackStitch(
  imageBuffer: Buffer,
  gridWidth: number,
  gridHeight: number,
): Promise<BackStitchSegment[]> {
  // Step 1: グレースケール化 + グリッドサイズにリサイズ
  const gray = await sharp(imageBuffer)
    .resize(gridWidth, gridHeight, { fit: 'fill' })
    .grayscale()
    .raw()
    .toBuffer({ resolveWithObject: true });

  const { data, info } = gray;
  const { width, height } = info;

  // Step 2: Sobel X
  const sobelX = [
    -1, 0, 1,
    -2, 0, 2,
    -1, 0, 1,
  ];
  // Step 3: Sobel Y
  const sobelY = [
    -1, -2, -1,
     0,  0,  0,
     1,  2,  1,
  ];

  // Step 4: エッジ強度マップを計算
  const edgeMap: number[] = new Array(width * height).fill(0);

  for (let y = 1; y < height - 1; y++) {
    for (let x = 1; x < width - 1; x++) {
      let gx = 0, gy = 0;
      for (let ky = -1; ky <= 1; ky++) {
        for (let kx = -1; kx <= 1; kx++) {
          const pixel = data[(y + ky) * width + (x + kx)];
          const ki = (ky + 1) * 3 + (kx + 1);
          gx += pixel * sobelX[ki];
          gy += pixel * sobelY[ki];
        }
      }
      edgeMap[y * width + x] = Math.sqrt(gx * gx + gy * gy);
    }
  }

  // Step 5: 閾値でエッジを二値化（強いエッジのみ抽出）
  const threshold = 80;
  const segments: BackStitchSegment[] = [];

  for (let y = 0; y < height - 1; y++) {
    for (let x = 0; x < width - 1; x++) {
      const strength = edgeMap[y * width + x];
      if (strength < threshold) continue;

      // 右方向のセグメント
      if (x + 1 < width && edgeMap[y * width + (x + 1)] >= threshold) {
        segments.push({
          x1: x, y1: y,
          x2: x + 1, y2: y,
          colorCode: strength > 150 ? 'DMC-310' : 'DMC-3799',
          ply: strength > 150 ? 2 : 1,
        });
      }
      // 下方向のセグメント
      if (y + 1 < height && edgeMap[(y + 1) * width + x] >= threshold) {
        segments.push({
          x1: x, y1: y,
          x2: x, y2: y + 1,
          colorCode: strength > 150 ? 'DMC-310' : 'DMC-3799',
          ply: strength > 150 ? 2 : 1,
        });
      }
    }
  }

  return segments;
}
```

## BackStitchSegment 型の確認・追加

packages/types/src/index.ts に BackStitchSegment が定義されているか確認。
なければ以下を追加:

```typescript
export interface BackStitchSegment {
  x1: number;
  y1: number;
  x2: number;
  y2: number;
  colorCode: string;
  ply: 1 | 2;
}
```

```typescript
import { detectBackStitch } from '@stitchlog/conversion-engine';

// extractColors の後
const backStitchSegments = await detectBackStitch(
  imageBuffer,
  patternData.stitchCountX,
  patternData.stitchCountY,
);

// レスポンスの layers.backStitch に設定
patternData.layers.backStitch = backStitchSegments;
```

## 制約
- ライブラリ追加なし（sharp のみ使用）
- セグメント数が 5000 を超える場合は threshold を 100 に上げて再計算する
- エラーが出たら止まって報告する
