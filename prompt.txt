## 想定される問題と対処

### セグメント数が5000超の場合
backstitchDetector.ts の threshold を 80 → 120 に変更する。
さらに超える場合は 150 まで上げる。

### それでも遅い場合
Sobel演算をグリッドサイズではなく
縮小画像（gridWidth/2 × gridHeight/2）で行い、
座標を2倍にスケールアップして返す。

```typescript
// リサイズを半分のサイズで行う
const halfW = Math.ceil(gridWidth / 2);
const halfH = Math.ceil(gridHeight / 2);

const gray = await sharp(imageBuffer)
  .resize(halfW, halfH, { fit: 'fill' })
  .grayscale()
  .raw()
  .toBuffer({ resolveWithObject: true });

// セグメント座標を2倍にスケール
segments.push({
  fromX: x * 2, fromY: y * 2,
  toX: (x + 1) * 2, toY: y * 2,
  ...
});
```

## 目標
- 処理時間: 2秒以内
- セグメント数: 1000〜3000程度
- データサイズ: 1MB以内
