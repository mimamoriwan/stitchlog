## Part B: PDFにバックステッチ線を描画

### 前提確認
generatePdf.ts を開いて以下の変数名を確認してから実装する:
- グリッド描画の開始X座標（offsetX相当）
- グリッド描画の開始Y座標（offsetY相当）
- 各マスのサイズ（cellSize相当）
変数名は必ず既存コードのものに合わせる。

### 実装仕様
layers.backStitch の各セグメント（fromX/fromY → toX/toY）を
グリッド上に線として重ねて描画する。

```typescript
// 描画ルール
for (const segment of patternData.layers.backStitch) {
  const x1 = offsetX + segment.fromX * cellSize + cellSize / 2;
  const y1 = offsetY + segment.fromY * cellSize + cellSize / 2;
  const x2 = offsetX + segment.toX   * cellSize + cellSize / 2;
  const y2 = offsetY + segment.toY   * cellSize + cellSize / 2;

  // plyCount で線幅を切り替え
  doc.setLineWidth(segment.plyCount === 2 ? 0.6 : 0.3);

  // colorCode で色を切り替え
  if (segment.colorCode === 'DMC-310') {
    doc.setDrawColor(0, 0, 0);
  } else if (segment.colorCode === 'DMC-3799') {
    doc.setDrawColor(43, 43, 43);
  } else {
    doc.setDrawColor(0, 0, 0); // フォールバック
  }

  doc.line(x1, y1, x2, y2);
}
```

### 挿入位置
グリッド描画ブロックの直後、フレンチノット描画の前に挿入する。
